## Tienda virtual

## ¿Como instalar el proyecto?

descarga el proyecto:

### `git clone` <br>

### `cd virtual-shop` <br>

### `pnpm i` <br>

## Scripts disponibles

En este proyecto tú puedes correr los siguientes comandos:

### `pnpm dev`

Esto lanzara el servidor de desarrollo que se encuentra ejecutando en la siguiente dirección

### http://localhost:5173
